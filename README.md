# kalijane
Kalijane website

#Installation

1 - Installer nodejs + npm
Télécharger l'installeur
https://nodejs.org/dist/v24.14.1/node-v24.14.1-x64.msi

Ouvrir sur le fichier et procéder à l'installation

2 - Installer Visual Studio Code (éditeur de texte)
https://code.visualstudio.com/download

3 - Télécharger le projet

Aller sur Github et sélectionner le projet Kalijane
Cliquer sur Code puis sur ouvrir avec Github Desktop
Installer Github Desktop
Ca devrait l'installer puis demander de get les fichiers (pull origin)

# Lancer le projet

1 - Aller dans l'explorateur Widonws et faire clic droit sur le dossier Kali jane puis Ouvrir avec Code (ou VS Code)

2 - Cliquer sur terminal puis New terminal

3 - Un terminal va apparaître en bas

4 - Pour la première fois, taper "npm install" puis touche entrée

5 - Ensuite à chauqe fois taper : npm run start

6 - Le site est accessible à l'adresse : http://localhost:3004/

# Faire des modifications

Le code est dans src

Dès que tu fait un changement et sauvegarde ça va se mettre à jour automatiquement dans le navigateur normalement.
